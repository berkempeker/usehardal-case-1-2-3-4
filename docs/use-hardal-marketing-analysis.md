# Hardal (usehardal.com) — Marketing Analysis

---

## What Does Hardal Do?

Hardal is a **server-side, first-party data platform** founded in 2023 in Istanbul, Turkey. It solves a critical problem: businesses lose 20-40% of their tracking data due to ad blockers, Safari ITP, iOS ATT, and the death of third-party cookies. Hardal moves data collection from the browser to the server, claiming **99% data accuracy**.

**Core offerings:**

- Server-side Google Tag Manager (sGTM) hosting
- Meta Conversions API (CAPI) & TikTok eAPI gateways
- Cookieless analytics dashboard
- Server-to-server (S2S) endpoints
- ETL/data governance (PII hashing, IP anonymization)
- Multi-cloud deployment (AWS, Azure, Huawei Cloud)

**Company timeline:**

- June 2022: Started as weekend project (originally web3 analytics)
- Feb 2023: Incorporated in Turkey
- Aug 2023: Pivoted to web2 server-side tracking & first-party data
- Nov 2023: First paying customer
- Jul 2024: $100K ARR, 5-person team
- Nov 2024: Pre-seed funding at $5M valuation (Startup Wise Guys + Berkay Mollamustafaoglu)
- Sep 2025: $500K ARR
- Feb 2026: Expanding to Berlin & Barcelona

## Competitive Landscape


| Competitor          | Positioning                       | Key Difference                       |
| ------------------- | --------------------------------- | ------------------------------------ |
| **Stape.io**        | Affordable sGTM hosting           | Hosting-only, no end-to-end platform |
| **Jentis**          | Enterprise privacy-first tracking | EU-focused, higher price point       |
| **Addingwell**      | sGTM hosting with extras          | Smaller feature set                  |
| **Taggrs.io**       | Budget sGTM hosting               | Limited integrations                 |
| **Tealium**         | Enterprise CDP + tag management   | Much more expensive, enterprise-only |
| **DIY sGTM on GCP** | Self-managed                      | Requires DevOps, $3,500-6,500+/mo    |


**Hardal's differentiation:** End-to-end platform (not just hosting), cookieless analytics included, multi-cloud options, Turkish/EU data residency, 1-3 day implementation vs months for DIY.
