## CASE 1: Cross-Domain Conversion Tracking

### Scenario:

A user visits our landing page at usehardal.com, clicks "Shopify App is now live" on the hero section, gets redirected to the Shopify App Store, and installs our app.

We need to track this complete journey without losing the user.

### Challenge:

How do you identify and track users who:

1. Visited usehardal.com landing page
2. Clicked through to the Shopify App Store
3. Successfully installed the Hardal app

### Your Task:

- Design the technical implementation (simple architecture diagram)

### Deliverables:

- Technical architecture document with data flow diagram
- Implementation code (JavaScript/API calls) with edge cases handled

- Month 1 vs Month 3 vs Month 6 strategy

- Success metrics and KPIs

### Deliverables:

- Budget allocation spreadsheet with channel strategy and expected metrics
- 90-day execution roadmap with risk analysis

## CASE 5: Automated Open Graph Image (OG) Generation

System (Mini Project)

### Scenario:

Hardal has 1,000+ blog posts. Each needs:

- Unique Open Graph images
- Consistent branding
- Automated generation
- Dynamic content (title, category, author)

### Challenge:

Build a web application that automatically generates Open Graph images for all blog posts.

### Your Task: Build a full-stack application deployed on Vercel that:

#### Features Required:

1. Input: Blog post metadata (title, author, category, date)
2. Image Generation:

- Use Hardal brand colors and fonts
- Include blog post title
- Add category badge
- Include Hardal logo
- Proper OG dimensions (1200x630px)

3. Automation:
- Bulk generation for multiple posts
- API endpoint for integration
- CSV upload for batch processing

4. Preview: Live preview before download
5. Export: Download individual or batch zip

#### Technical Requirements:

- Deploy on Vercel
- Use any tech stack (Next.js recommended)

### Deliverables:

- Live Vercel URL + GitHub repository with comprehensive README
- Optional: Loom video walkthrough (5 min max)
- Hint: You can use Vercel's OG Image Generation https://vercel.com/docs/og-image-generation
- Hint: Do not overthink, this is a small project.
